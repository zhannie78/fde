## Unstyled-link contract (applies to ALL six links in this plan)

Every link added here MUST be visually invisible as a link. Wrap the target text in a
`next/link` `Link` with exactly:

```
className="text-inherit no-underline"
```

- `text-inherit` makes the anchor inherit the surrounding text color (so a phrase that
  is currently `text-primary` stays `text-primary`; a phrase that is plain foreground
  stays foreground).
- `no-underline` removes the default underline.
- Do NOT add any `hover:`, `focus:`, `visited:`, `transition`, or color utility of any
  kind. No hover state, no focus color change, no visited color change.
- When the phrase already sits inside a styled element (`<span className="text-primary">`
  or `<strong className="lede-highlight">`), place the `Link` INSIDE that element and wrap
  only the text â€” this preserves the existing appearance while `text-inherit` picks it up.
