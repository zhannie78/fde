# Sketch Manifest

## Design Direction

Complete visual system replacement for the AI Deployed landing page. Moving away from the warm-neutral editorial palette (paper/ink-navy/signal-green, Fraunces + IBM Plex Sans) toward a modern, high-impact **developer-portfolio** aesthetic for a solo, named AI-native forward-deployed engineer/consultant (Annie). Core requirements: monospace type as the primary voice (not just for code snippets), pixelated/retro-tech visual texture, and a personal, first-person hero ("Hello, I'm Annie, a forward deployed engineer and ai consultant") replacing the prior anonymous-FDE framing. Content stays the same (gap → fix → outcomes → offer → CTA, ROI calculator, pricing) — this is a visual-system swap, not a copy rewrite.

## Reference Points

- Wix Studio "Transparent Video" inspiration page — bold high-contrast type, scroll-based modular card sections, generous negative space, vibrant gradient accents punctuating neutral/dark backgrounds, minimalist-but-experimental mood. Translated here into a mono/pixel developer register rather than copied literally.
- User's explicit anchors: "very modern and impressive," "mono style font," "pixelated looks," "developer" feel, "freelance developer portfolio."
