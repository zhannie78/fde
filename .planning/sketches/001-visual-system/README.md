# Sketch 001: Visual System

## Design Question
What full visual system (typography, color, texture, layout mood) should replace the current warm-neutral editorial look, for a mono/pixel developer-portfolio aesthetic introducing Annie as a named forward-deployed engineer / AI consultant?

## How to View
open .planning/sketches/001-visual-system/index.html

## Variants

- **A: Terminal // CRT** — near-black background, phosphor-green primary with amber accent, scanline + vignette texture, terminal window chrome (traffic-light dots, `annie@fde:~$` prompts), blinking cursor, dashed ASCII-style dividers. Nostalgic hacker-terminal register — leans hardest into "developer," most raw/technical mood.
- **B: Pixel-Grid Brutalist** — light background with a faint pixel grid, thick black borders, hard offset drop-shadows (no blur), one loud accent (electric blue) plus acid-lime highlight, big blocky uppercase headlines, pixel-stepped card corners. Daylight, confident "modern portfolio" register — closest to the Wix inspiration's bold-type + generous-space energy, least "hacker," most "impressive design portfolio."
- **C: Dark Neon IDE** — deep charcoal/navy background, syntax-highlighting-inspired accent trio (violet/cyan/pink) used sparingly like editor tokens, glass-panel cards with blur, code-editor tab bar + decorative line-number gutter on the hero, gradient-text headline accents. Premium dark-SaaS-meets-IDE register — most "polished/modern," least raw-nostalgic.

## Winner

**B — Pixel-Grid Brutalist.** Confirmed by user 2026-07-20.

Post-selection refinements applied directly to variant B:
- Added a new "The Access Gap" full-bleed dark band between Hero and The Fix — explicit enterprise-vs-SMB market framing ("Enterprises pay a premium to embed AI-native engineers like this. Now small businesses can too."), pulled from PIVOT-BRIEF's market thesis but not previously surfaced as on-page copy.
- Pricing revised: setup **under $5k** (was $10k) with an hourly-rate alternative mentioned; retainer **under $1,000/mo** (was $2k/mo). PIVOT-BRIEF.md updated to match — these are now the locked figures.

## What to Look For

- Which register feels right for a solo consultant's personal brand: raw-hacker (A), confident-portfolio (B), or premium-SaaS (C)?
- Legibility and impact of the mono display type at hero scale in each palette.
- Whether the pixel/texture treatment (scanlines, grid, dot-noise) reads as "impressive" or "too busy" once real content is present.
- The stat cards / offer cards / process list — same content, three different card treatments (bordered-dark, brutalist-shadow, glass-blur). Which supports the pricing/outcomes numbers best?
- Try the viewport buttons (bottom-right) to check mobile width.
- Cherry-pick freely — e.g. "B's layout and card shapes with A's terminal chrome" is a valid answer.
